# 音乐播放器 UI 原型设计

> 技术栈：SwiftUI | 平台：iOS 15+ | 设计风格：现代简约

---

## 一、整体设计风格

### 设计原则
- **现代简约**：大量留白，清晰的视觉层次
- **毛玻璃效果**：使用 SwiftUI 的 `.ultraThinMaterial` 营造层次感
- **动态交互**：平滑的动画过渡，手势友好
- **深色/浅色主题**：支持系统主题自动切换

### 主色调
```
主色调: #FF6B6B (珊瑚红) - 用于播放按钮、进度条、高亮元素
背景色: 深色模式 #1A1A2E / 浅色模式 #F5F5F7
辅助色: #4ECDC4 (青绿) - 用于歌词高亮
文字色: 深色模式 #FFFFFF / 浅色模式 #1A1A2E
```

## 四、交互动画

### 4.1 播放界面过渡
```
迷你播放器 ──(上滑)──> 全屏播放界面
全屏播放界面 ──(下滑)──> 返回 + 迷你播放器
```

### 4.2 歌词滚动效果
- 当前播放行：字体放大 + 颜色高亮 + 居中显示
- 已播放行：透明度降低
- 未播放行：正常显示
- 滚动动画：平滑滚动到当前行

### 4.3 专辑封面动画
- 播放状态：缓慢旋转（30秒/圈）
- 暂停状态：停止旋转
- 切歌时：缩放动画 + 新封面淡入
